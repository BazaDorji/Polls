== Description ==
The Simple Polls Plugin adds a polling feature to your WordPress site, allowing visitors to vote on various topics and see the results. Easily create polls through the WordPress admin panel and embed them in posts or pages using shortcodes.

== Installation ==
1. Upload the `simple-polls-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a new poll by navigating to `Polls > Add New` in the WordPress admin dashboard.
4. Enter your poll question and options in the provided fields.
5. Embed the poll in a post or page using the `[spp_poll id="X"]` shortcode, where `X` is the ID of the poll.
6. View the poll on your site, allowing users to vote and see results.

== Frequently Asked Questions ==

= How do I create a poll? =
After activating the plugin, go to `Polls > Add New` in your WordPress admin dashboard. Enter your poll question and options, then publish the poll. Use the generated shortcode to embed the poll in any post or page.

= Can I display multiple polls on the same page? =
Yes, you can display multiple polls on the same page by embedding multiple shortcodes, each with the ID of the respective poll.

= How do users vote in a poll? =
Users can vote by selecting an option from the poll and clicking the "Vote" button. The results will be displayed immediately after voting.

= Can I see how many votes each option received? =
Yes, the plugin displays the vote count for each option below the poll. You can also view vote counts in the WordPress admin by editing the poll.
